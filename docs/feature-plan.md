# Delivery Flow (1 Month Plan)

Implement one feature at a time in this order:

1. Expert Profile (done)
2. Expert listing and search
3. Doubt posting
4. Chat/session screen
5. Wallet and transactions
6. Booking/schedule flow
7. Notifications and history
8. Authentication and authorization (last)

For each feature, keep this cycle:

1. SQL tables
2. Backend API (route -> controller -> service -> repository)
3. Frontend page/components
4. API integration
5. Test flow manually
